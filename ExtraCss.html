PSEUDO ELEMENTOS Y PSEUDOCLASES

Una pseudoclase en CSS es una extensión del selector que te permite aplicar estilos a un elemento en función de su estado o relación con el documento HTML. No se enumeran como los selectores regulares, sino que se utilizan para definir estilos específicos para estados particulares de los elementos, como :hover para cuando el mouse está sobre el elemento, :focus para cuando el elemento está seleccionado, :active para cuando está siendo activado, entre otros. Esto permite crear estilos dinámicos y mejorar la experiencia del usuario en la interfaz web.(Solo funcionan en elemento en bloque)

::first-line

La pseudoclase ::first-line es una pseudo-clase de CSS que te permite aplicar estilos específicos al primer línea de un elemento de bloque. Esto significa que puedes cambiar el formato de la primera línea de un párrafo, un encabezado u otro elemento de bloque sin afectar al resto del texto dentro del mismo elemento. Por ejemplo, puedes utilizar ::first-line para cambiar la fuente, el tamaño de fuente, el color o la decoración del texto solo en la primera línea de un párrafo. Esto es útil para resaltar visualmente el comienzo de un párrafo o para hacer que ciertas partes del texto se destaquen..


::PlaceHolder

La pseudoclase ::placeholder en CSS se utiliza para estilizar el texto de marcador de posición dentro de un elemento de formulario, como un campo de entrada (input) o un área de texto (textarea). Este texto de marcador de posición es el texto que aparece dentro del campo antes de que el usuario ingrese algún valor.
